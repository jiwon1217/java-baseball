## 기능 목록

-[x] 사용자가 맞춰야 하는 수 생성
    -[ ] 숫자를 3개만 생성
    -[ ] 1이상 9이하의 숫자만 생성
    -[ ] 생성 된 수는 중복되지 않아야 함
-[ ] 게임 시작 문구 출력
-[ ] 숫자 입력 문구 출력
-[ ] 사용자의 숫자 입력
-[ ] 입력한 수에 대한 검증
    -[ ] 숫자를 3개만 입력
    -[ ] 1이상 9이하의 숫자만 입력
    -[ ] 입력한 수는 중복되지 않아야 함
-[ ] 입력한 수에 대한 결과를 표시
    -[ ] 낫싱 여부 판단
    -[ ] 3스트라이크 여부 판단
    -[ ] 스트라이크, 볼 여부 판단
-[ ] 게임 종료 문구 출력
-[ ] 재시작 문구 출력
-[ ] 재시작 및 종료에 대한 처리
